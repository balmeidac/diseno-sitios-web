# Landing Page - Semana 05

## Descripción
Landing page profesional de **TechSolutions** desarrollada con HTML5 y CSS3, siguiendo los fundamentos y buenas prácticas de desarrollo web.

## Estructura del Proyecto

```
semana-05/
├── index.html          # Página principal (estructura HTML5 semántica)
├── css/
│   └── style.css       # Estilos CSS3 con variables y responsive design
├── img/                # Carpeta para imágenes (futura expansión)
└── README.md           # Este archivo
```

## Características HTML5

### Elementos Semánticos Utilizados:
- **`<!DOCTYPE html>`** - Declaración correcta del documento
- **`<html lang="es">`** - Atributo de idioma
- **`<meta>`** - Metadatos esenciales (charset, viewport, descripción)
- **`<header>`** - Encabezado de la página con logo y navegación
- **`<nav>`** - Navegación con roles de accesibilidad
- **`<main>`** - Contenido principal de la página
- **`<section>`** - Secciones temáticas (hero, about)
- **`<article>`** - Contenido de artículo en la sección about
- **`<footer>`** - Pie de página con información y contacto

## Características CSS3

### Estilos Implementados:
- **CSS Variables** - Colores y estilos reutilizables
- **Flexbox y Grid** - Layouts modernos y responsivos
- **Gradientes** - Fondos atractivos
- **Transiciones y Animaciones** - Efectos suaves
- **Media Queries** - Diseño responsivo para móviles y tablets
- **Scroll Suave** - Comportamiento de navegación fluido

## Secciones de la Página

### 1. **Header (Encabezado)**
- Logo con tagline
- Navegación principal con enlaces internos
- Efecto hover con subrayado animado
- Posición fija para acceso rápido

### 2. **Hero Section**
- Título principal atractivo
- Descripción del servicio
- Botón CTA (Call-to-Action) interactivo
- Fondo con gradiente azul

### 3. **About Section**
- Descripción de la empresa
- Lista de características con iconos
- Diseño limpio y legible

### 4. **Footer**
- Información de la empresa
- Enlaces rápidos
- Datos de contacto (email y teléfono)
- Copyright

## Responsiveness

La página está optimizada para:
- **Desktop** (1200px+)
- **Tablet** (768px - 1199px)
- **Mobile** (480px - 767px)
- **Small Devices** (-480px)

## Accesibilidad

Implementadas características WCAG:
- Atributos `role` semánticos
- `aria-label` en enlaces importantes
- Estructura HTML clara y lógica
- Contraste de colores adecuado
- Enlaces con hover visible

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. Navega entre secciones usando el menú superior
3. El sitio se adapta automáticamente a tu dispositivo
4. Los efectos hover aparecen al pasar el mouse sobre los elementos

## Tecnologías Utilizadas

- **HTML5** - Marcado semántico
- **CSS3** - Estilos avanzados, variables, flexbox, grid
- **Responsive Design** - Mobile-first approach
