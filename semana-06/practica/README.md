# Práctica de Estructura HTML Básica

## Descripción
Este ejercicio de práctica tiene como objetivo aplicar los conceptos básicos de estructura HTML aprendidos en clase. Se ha creado una página web simple que incluye diferentes elementos estructurales y de contenido de HTML.

## Contenido del Proyecto
El archivo `index.html` contiene una página web básica con los siguientes elementos:

### Estructura Principal
- Encabezado (`<header>`) con el título principal
- Contenido principal (`<main>`) que incluye:
  - Imagen del logo de la UTEQ
  - Títulos jerárquicos (`<h1>`, `<h2>`, `<h3>`)
  - Párrafos de texto (`<p>`)
  - Lista ordenada (`<ol>`) con los contenidos del libro
  - Lista desordenada (`<ul>`) con tecnologías front-end
  - Lista de definición (`<dl>`) que describe las partes de un documento HTML
- Pie de página (`<footer>`) con información de derechos de autor

### Elementos Utilizados
- Encabezados: `<h1>`, `<h2>`, `<h3>`
- Párrafos: `<p>`
- Listas:
  - Ordenada: `<ol>`, `<li>`
  - No ordenada: `<ul>`, `<li>`
  - De definición: `<dl>`, `<dt>`, `<dd>`
- Imagen: `<img>`
- Elementos semánticos: `<header>`, `<main>`, `<footer>`

## Objetivos de Aprendizaje
Al completar esta práctica, los estudiantes podrán:
1. Crear una estructura básica de documento HTML5
2. Utilizar correctamente los elementos semánticos de HTML5
3. Implementar diferentes tipos de listas
4. Insertar imágenes en una página web
5. Organizar el contenido de manera jerárquica

## Instrucciones
1. Abre el archivo `index.html` en tu navegador web para ver el resultado
2. Examina el código fuente para entender la estructura
3. Realiza las modificaciones que consideres necesarias para practicar
